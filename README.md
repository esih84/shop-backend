# ESI E-Commerce Backend

High-performance e-commerce backend built with NestJS, PostgreSQL, Redis, and BullMQ.

## Features

- **Auth**: OTP-based phone authentication (JWT access + refresh tokens)
- **Products**: Categories (tree), products, variants, images, attributes, discounts
- **Reviews**: Product reviews with approval workflow
- **Cart**: Session cart with stock validation
- **Orders**: Transactional order creation with coupon + loyalty point deduction
- **Loyalty**: Tiered loyalty program (Bronze/Silver/Gold/Platinum) with point earn/redeem
- **Coupons**: Percentage/fixed/free-shipping coupons with usage limits
- **Wishlist**, **Referrals**, **Blogs**, **Banners**, **Collections**, **Analytics**
- **Upload**: S3 + Sharp image processing (thumbnail/medium/large/original WebP)
- **RBAC**: USER / ADMIN roles via JWT guards
- **Caching**: Redis-backed (products cached 1hr)
- **Swagger** at `/api/docs`

## Quick Start

```bash
npm install
cp .env.example .env   # fill in your values
npm run start:dev
```

### Seed

```bash
npx ts-node -r tsconfig-paths/register src/database/seed.ts
```

Seeds: Loyalty tiers (Bronze/Silver/Gold/Platinum) + admin user (phone: `09000000000`).

## Project Structure

```
src/
├── common/       decorators, filters, guards, interceptors
├── config/       TypeORM, Redis, JWT, AWS
├── database/     seed.ts
└── modules/      auth, users, categories, products, reviews,
                  cart, orders, blogs, banners, collections,
                  loyalty, coupons, wishlist, referrals,
                  analytics, upload, campaigns
```
